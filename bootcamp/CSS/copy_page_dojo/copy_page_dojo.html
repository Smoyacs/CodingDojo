JavaScript Basics

JavaScript is a cross-plataform, object-oriented scripting language. JavaScript is extremly popular for a variety of reasons. It is a small and lightwieght language allowing maximun flexibility for developers to take it int bunch of different directions. JavaScript lives inside a host environment (a web browser or node server), it can be connect to the objects of these environments to provide programmatic control over them.

  • Variable Declaration JavaScript variables are containers for storing data values - imagine a cup you fill with coffe , the cup holds the coffe, a variable holds a value. All JavaScript variables must be identifed with uniques names. These uniques names are called identifiers. var x;

  • Variable Assignment Assignment operatos assign values to JavaScript variables - our cup can now have coffe poured in it, giving our variable a value to hold.The = assignment operator assigns a value to a variable. var x = 10;

  • Data types Data types are an important concept; to be able to operate on variable you need to know the data type. There are six data types that are JavaScript primitives: Boolean - True or false;; null - null aka nothing; Number - 42 or 3.14159; string - "Coding Dojo Rocks!"; Array - [1, 'Coding',2,'Dojo'] and Object - {first_name:'Jane',last_name:'Doe'}

  • Conditionals When you write code, you want to perform different actions for different decisions - hitting different code blocks based on values or conditions that have been met. You can use conditional statements in roy code to accomplish this. There are the following conditionals statements: if a especified conditions is true, do this code in our code block; else if to specify a new conditions to test, if the first_name conditions is false; else we execute this block of code;

  • Loops There are many different kinds of loops in every programming language, but they all essentially do the same thing: the will repeat an actions some number of times. Imagine you have to run a mile, well you run arounf the track four times and then you stop. Thats a loop!

  • Function Functions are an encaspulation of a code block. When we call our function this will run that code block. Think of it as a list of intructions.
